## 진행

0. 모이는 시간 / 요즈음 프로그래밍 뭐하나(18:00 ~ 18:15)

1. LT : 알고리즘(테마)에 대해서 발표자가 준비 (18:15 ~ 18:45)

    - [스파게티 code C++](http://www.prefield.com/algorithm/)
